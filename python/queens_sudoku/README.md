# 皇后数独游戏辅助脚本

## 环境要求
- Windows 11 系统
- Python 3.12+
- 使用 uv 进行环境管理

## 安装依赖

```bash
python -m uv sync
```

## 使用方法

### 1. 正常运行模式

确保游戏窗口已打开，标题为"智商不够别点"，然后运行：

```bash
# 单牛模式
python -m uv run main.py

# 双牛模式
python -m uv run main.py --2
```

## 功能说明

1. **窗口查找**：自动查找标题为"智商不够别点"的游戏窗口
2. **内存截图**：使用 pywin32 进行高效内存截图
3. **游戏区域定位**：通过颜色检测自动定位游戏操作区
4. **棋盘提取**：识别并提取棋盘色块信息
5. **数独求解**：使用回溯法求解皇后数独问题
6. **彩色打印**：在终端中彩色显示棋盘和解决方案
7. **自动点击**：自动解答（支持后台窗口）

## 游戏规则（单牛模式）

- 每个颜色有且只有一头牛
- 每行有且只有一头牛
- 每列有且只有一头牛
- 每头牛周围8个格子内没有牛

## 游戏规则（双牛模式）

- 每个颜色有且只有两头牛
- 每行有且只有两头牛
- 每列有且只有两头牛
- 每头牛周围8个格子内没有牛

## 输出说明

1. **棋盘打印**：彩色显示提取的棋盘
2. **文字形式解**：显示每头牛的位置
3. **棋盘形式解**：在棋盘上用🐮标记牛的位置

## 注意事项

1. 确保游戏窗口处于活动状态
2. 如果无法定位游戏区域，可能需要调整游戏窗口大小或分辨率
3. 脚本已启用 DPI 感知，支持高分辨率屏幕
